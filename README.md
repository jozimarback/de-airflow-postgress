# DE Airflow Postgress

### Instructions

Run docker compose with the following command:

> docker-compose up -d

### Tools

- Postgres
    - pgadmin [http://localhost:5050/]
- Airflow
    - airlfow [http://localhost:8080]
    - flower [http://localhost:5555]
- Spark
    - master [http://localhost:9080]
